## Part one - basics

In this project, we created a _bulletin board_, also called _forum_, using `PHP` & `MySQL`.  
We used `SASS` & `Bootstrap`.  
The tooling used `docker` & `heroku`.

### Features

We designed a database to handle four types of data:

- users
- boards
- topics
- messages
